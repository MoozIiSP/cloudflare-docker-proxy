快捷拉取命令

也可以通过前缀方式，直接从不同仓库拉取镜像：

# 拉取 Docker 官方镜像
docker pull docker.{{host}}/library/nginx:latest

# 拉取 Quay 镜像
docker pull quay.{{host}}/coreos/etcd:latest

# 拉取 GCR 镜像
docker pull gcr.{{host}}/google-containers/busybox:latest

# 拉取 GitHub 容器镜像
docker pull ghcr.{{host}}/github/super-linter:latest
